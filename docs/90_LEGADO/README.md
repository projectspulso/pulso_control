# Legado - Ponto de Entrada

Esta pasta guarda o acervo antigo que ainda pode ajudar, mas nao deve ser tratado como estado atual sem confronto com o codigo.

## Ler primeiro

1. `MAPA_DA_DOCUMENTACAO_ANTIGA.md`

## Estrutura

- `analises/` -> analises antigas e snapshots de implementacao
- `checklists/` -> checklists de fase e operacao
- `operacao/` -> guias operacionais pontuais
- `relatorios/` -> relatorios de testes e validacao
- `status_e_resumos/` -> status historicos e resumos
- `logs/` -> artefatos de log

## Regra

Se um documento desta pasta contrariar `app/`, `lib/`, `components/` ou os docs curados de `docs/00_MESTRE` a `docs/50_BLUEPRINTS`, o legado perde.
