# Produto - Fonte de Verdade

Esta pasta passa a ser a camada oficial de produto do projeto.

## Tese oficial

`PULSO Control` nao e uma "maquina de canais dark multi-rede".

`PULSO Control` e um sistema operacional para videos curtos faceless originais, com pipeline editorial, producao assistida, publicacao assistida e humano no loop.

## Como usar esta pasta

- `01` a `04` definem categoria, publico e problema
- `05` e `06` definem o produto atual e o fluxo real
- `07` a `10` definem execucao, prioridade e decisao
- `11` traduz isso em linguagem de pitch e mensagem
- `12` consolida tudo em formato de handoff
- `13` e `14` ajudam onboarding e alinhamento de linguagem
- `15` fecha a passagem de documento para execucao
