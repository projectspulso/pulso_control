📚 3. Tipos de Conteúdos (já prontos para transformar em workflow)

1. Conteúdo Narrado Curto (Storytelling)

45–60 segundos

História rápida, “moral da história”, micro contos

Alta retenção

2. Curiosidades e fatos

15–30 segundos

“Você sabia que…”

Muito fácil escalar com IA

3. Psicologia humana

20–50 segundos

“Se você faz isso, cuidado...”

Um dos nichos mais virais de shorts

4. Motivacional / Reflexão

Texto + trilha + imagens de IA

Feito para viralizar no TikTok e Kwai

5. Contos de Mistério / História real

Leve suspense

Alto watchtime

Formato ideal para ter engajamento no YouTube

🔁 4. Workflow Completo n8n para Produção em Massa
(1) Lista de ideias no Supabase

→ tabela: conteudo.ideias

(2) n8n puxa 1 ideia → gera roteiro

→ ChatGPT gera roteiro curto
→ salva em conteudo.roteiros

(3) n8n chama TTS

Voz neural (Google / ElevenLabs / opcional chinesa)

(4) n8n chama IA de vídeo

(CapCut template via automação)

(Runway / Pika / Kling quando viável)

(5) n8n gera 4 versões do mesmo vídeo

9:16 — Shorts

9:16 — TikTok

9:16 — Reels

9:16 — Kwai (legenda maior)

(6) n8n publica automaticamente (ou deixa no buffer)

Salva os links

Registra em Supabase

(7) n8n puxa métricas

Views

Retenção

CTR

Assinantes ganhos

(8) IA recalibra conteúdos

Decide o que repetir

O que aumentar

O que parar
