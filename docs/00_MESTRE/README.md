# Mestre - Ponto de Entrada

Esta pasta guarda os documentos que organizam o pensamento do projeto antes de entrar em setup, banco, automacao ou legado.

## Ler primeiro

1. `TRILHA_OFICIAL_DE_EXECUCAO.md`
2. `CHECKLIST_DA_TRILHA_OFICIAL.md`
3. `ESTADO_ATUAL_E_FONTES_DE_VERDADE.md`

## Estrutura

- `TRILHA_OFICIAL_DE_EXECUCAO.md` -> ordem oficial para ler, validar e executar
- `CHECKLIST_DA_TRILHA_OFICIAL.md` -> lista rapida para o humano marcar a execucao
- `ESTADO_ATUAL_E_FONTES_DE_VERDADE.md` -> o que e fonte de verdade hoje
- `00_MAPA_COMPLETO_PULSO.md` -> mapa estrutural amplo do sistema
- `basedo_app.md` -> resumo antigo do app real
- `FLUXO_PRODUCAO_COMPLETO.md` -> fluxo editorial completo

## Regra

Se houver conflito entre qualquer doc operacional e os documentos desta pasta, primeiro alinhar a decisao aqui e so depois executar.
