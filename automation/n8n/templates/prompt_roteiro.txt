# TAREFA
Crie um roteiro completo para um vídeo curto (30-60 segundos) seguindo esta estrutura:

## HOOK (3 segundos)
- Frase de impacto inicial que prenda atenção imediatamente
- Use dado chocante, pergunta intrigante ou afirmação controversa

## DESENVOLVIMENTO (40 segundos)
- 3-4 pontos principais
- Cada ponto com fatos concretos e dados verificáveis
- Linguagem acessível mas intrigante
- Mantenha ritmo dinâmico

## CONCLUSÃO (7 segundos)
- Resumo impactante
- Call-to-action (seguir, comentar, compartilhar)
- Frase de efeito final

## EXTRAS (Sugestões de Produção)
- **B-roll sugerido**: Descrição de imagens/vídeos de apoio
- **Tom de voz**: Como narrar (misterioso, dramático, entusiasta, etc.)
- **Música**: Estilo de trilha sonora
- **Efeitos**: Transições, zooms, legendas especiais
